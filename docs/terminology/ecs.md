# ECS - Experimentation and Configuration Service

Microsoft's feature flag and configuration system for Teams. Controls codec selection, ICE behavior, SRTP settings, transport options, and audio platform configuration.

## Structure

Configuration delivered as JWT-like structure with 300+ keys.

## Examples

- Codec enablement flags
- ICE timeout values
- SRTP cipher preferences
- Audio processing parameters
