# SDES - Source Description

RTCP packet type containing participant identification information such as canonical name (CNAME).

## Items

- **CNAME** - Canonical name (required)
- **NAME** - User name
- **EMAIL** - Email address
- **PHONE** - Phone number
- **LOC** - Geographic location
