# SILK - SILK Audio Codec

Audio codec originally developed by Skype. Teams' primary production codec.

## Variants

| PT | Name | Sample Rate | Bitrate |
|----|------|-------------|---------|
| 103 | SILK | 8000 Hz | Narrowband |
| 104 | SILK | 16000 Hz | 16-40 kbps (primary) |

## Features

- Variable bitrate (VBR)
- In-band FEC (`useinbandfec=1`)
- Low latency
- Optimized for speech
