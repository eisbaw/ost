# VQE - Voice Quality Enhancement

Audio processing pipeline combining multiple algorithms to improve call quality.

## Components

- [AEC](aec.md) - Acoustic Echo Cancellation
- [AGC](agc.md) - Automatic Gain Control
- [VAD](vad.md) - Voice Activity Detection
- Noise Suppression
- De-reverberation
