# PCMU - PCM mu-law

ITU-T G.711 audio codec using mu-law companding. North American telephone standard.

## Characteristics

- **Payload Type**: 0
- **Sample Rate**: 8000 Hz
- **Bitrate**: 64 kbps (fixed)
- **Quality**: Narrowband telephone quality
