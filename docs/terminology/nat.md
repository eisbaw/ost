# NAT - Network Address Translation

Network technique that maps private IP addresses to public IP addresses. Creates challenges for peer-to-peer communication.

## NAT Types

- **Full Cone** - Most permissive
- **Restricted Cone** - Filters by IP
- **Port Restricted Cone** - Filters by IP and port
- **Symmetric** - Most restrictive

## Traversal

ICE, STUN, and TURN work together to establish connections through NAT.
