# E2EE - End-to-End Encryption

Encryption where only the communicating parties can decrypt content. The service provider cannot access the plaintext.

## Teams Support

Teams supports E2EE for 1:1 calls when enabled by admin policy.

## Difference from SRTP

- **SRTP** - Encrypts media between endpoints and servers (hop-by-hop)
- **E2EE** - Encrypts media only between participants (end-to-end)
