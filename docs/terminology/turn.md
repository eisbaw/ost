# TURN - Traversal Using Relays around NAT

Relay protocol (RFC 5766) for NAT traversal when direct peer-to-peer connection fails. Traffic is relayed through a TURN server.

## Teams TURN Service

FlightProxy at `api.flightproxy.teams.microsoft.com:3478`

## Authentication

TURN servers typically require credentials, provided via ICE candidate attributes.
