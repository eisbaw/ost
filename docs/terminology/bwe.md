# BWE - Bandwidth Estimation

Algorithms that estimate available network bandwidth for adaptive bitrate control.

## Algorithms

- **GCC** - Google Congestion Control (delay-based)
- **REMB** - Receiver Estimated Maximum Bitrate
- **Transport-CC** - Transport-wide Congestion Control

## Purpose

Adapt video/audio bitrate to available bandwidth to minimize:
- Packet loss
- Latency
- Jitter
