# Graph API - Microsoft Graph API

RESTful API for accessing Microsoft 365 services including Teams chat, user information, calendar, and more.

## Endpoint

```
https://graph.microsoft.com/v1.0/
https://graph.microsoft.com/beta/
```

## Common Endpoints

- `/me` - Current user profile
- `/me/chats` - User's chats
- `/me/presence` - User's presence status
- `/teams` - Teams and channels
