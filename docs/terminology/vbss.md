# VBSS - Video-Based Screen Sharing

Teams screen sharing implemented as a separate video stream with optimized encoding for screen content.

## Characteristics

- Separate SDP media line from camera video
- H.264 Constrained Baseline profile
- 5-15 fps (lower than camera video)
- Optimized for text and static content
