# SSRC - Synchronization Source

32-bit identifier for an RTP stream source. Randomly generated to be unique within a session.

## Usage

- Identifies the source of an RTP stream
- Used in RTCP reports to reference specific streams
- Multiple SSRCs can exist in a single session (e.g., audio and video)
